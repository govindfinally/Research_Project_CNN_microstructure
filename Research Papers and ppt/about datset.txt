Dataset Details — UHCS (Ultrahigh Carbon Steel) Microstructure
Overview (DeCost, Francis & Holm — Acta Materialia, 2017)

The dataset includes 961 SEM (scanning electron microscopy) micrographs of UHCS, spanning different heat treatments, magnifications, and contrast modes (SE and BSE).

arXiv
+1

Each image has a label indicating the primary microconstituent, such as:

spheroidized cementite

carbide network

pearlite (and variants like pearlite + spheroidite, Widmanstätten cementite, martensite/bainite)

Image Resolution & Preprocessing

Original resolution: approx. 645 × 484 pixels.

The authors also generated a larger dataset of cropped 224 × 224 sub-images, yielding 2400 images, to train CNN models.

Accessibility

Data portal: The dataset is hosted on NIST materials data repository, accessible via a handle like materialsdata.nist.gov/.../11256/940.

A companion GitHub repository contains a download script (download.sh) that retrieves:

microstructures.sqlite (metadata)

micrographs.zip (the micrograph images)

GitHub

Use Cases Demonstrated

Classification tasks:

Classifying the primary microconstituent (e.g., spheroidite vs network vs pearlite), achieving high accuracy (>95%) using standard CNN feature extraction.
